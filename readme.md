# email validator

## what it does

- look up MX record us
- checks if the domain name is a temporary domain address from this [list](https://github.com/disposable/disposable)

## how to run locally

`npm install`

`npm run dev`

### running tests

`npm run test`

then you should be able to open http://localhost:3000/api-docs/
